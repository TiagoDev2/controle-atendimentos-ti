# Controle de Atendimentos TI - v2

Projeto pessoal desenvolvido com HTML, CSS e JavaScript para simular uma rotina simples de atendimento interno de TI.

## Funcionalidades
- cadastro de novos atendimentos
- edição de registros
- atualização de status
- filtro por situação
- busca por nome, setor, tipo ou descrição
- exportação em PDF
- exportação em CSV
- armazenamento local com localStorage

## Tecnologias usadas
- HTML
- CSS
- JavaScript puro
- html2pdf.js

## Objetivo do projeto
A ideia é  praticar organização de dados, manipulação do DOM e construção de uma interface simples voltada para uma situação comum da área de suporte técnico.

## Possíveis melhorias futuras
- autenticação de usuário
- integração com banco de dados
- painel com gráficos
- histórico de alterações
